# m-loot

Pedal controlador MIDI para tocar con _looper_ basado en un DAW como Ableton Live o Mobius.

## Objetivos

Crear una pedalera MIDI con las siguientes características:

- **10 footswitches** programables (pulsadores de pie)
- **5 encoders (knobs) rotativos** para controles continuos
- **Pantalla OLED de 0.96"** para visualización de estado (128x64 píxeles)
- **8 LEDs indicadores** de estado
- **Control MIDI USB plug-and-play** (sin drivers adicionales)
- **Comunicación MIDI completa** con Ableton Live
- **Tempo tap visible** en pantalla
- **Presupuesto total: 100-150€**

### Funcionalidades principales

1. Lanzar/parar escenas en Ableton Live
2. Trigger de clips y pistas
3. Control de parámetros de efectos (knobs)
4. Pantalla mostrando:
   - Escena actual
   - Tempo BPM
   - Estado de pistas (grabando, reproduciendo, etc.)
   - Parámetros de efectos en tiempo real
   - Indicadores visuales de footswitches activos
