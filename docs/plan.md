# Cronograma de Implementación

**Duración total:** 5-6 semanas (incluyendo envíos).   
**Horas de trabajo:** 40-50 horas

## Desglose por semana

- **Semana 1:** Investigación, planificación, compras (9 horas)
- **Semana 2:** Recepción de componentes, setup Arduino IDE (3.5 horas)
- **Semana 3:** Testing en protoboard de cada componente (10 horas)
- **Semana 4:** Soldadura y ensamblaje en caja (12 horas)
- **Semana 5:** Programación y optimización (8 horas)
- **Semana 6:** Mapeo MIDI en Ableton Live y pruebas finales (7.5 horas)

### Semana 1: Investigación y compras

#### Paso 1: Recopilar información (2h)

- Revisar especificaciones de pedales comerciales similares
- Estudiar tutoriales Arduino MIDI en YouTube
- Revisar documentación MIDIUSB library

#### Paso 2: Diseñar mi versión 

- Decidir número de footswitches (recomendado: 10)
- Decidir número de encoders (recomendado: 5)
- Diseñar layout en papel o digital
- Calcular tamaño de caja necesaria

### Paso 3: Comprar componentes (1h)

- Utilizar lista de materiales del documento principal
- Presupuesto total: 126€ (típico)
- Proveedores: Amazon (Arduino rápido), Aliexpress (componentes baratos)

Nota: Aliexpress tarda 14-21 días. Usar Amazon Prime si urgencia.
